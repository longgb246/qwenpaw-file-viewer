# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-03

### Added
- Initial release
- Backend HTTP API server (port 39150) for file reading
- Frontend React component with inline file preview panel
- Markdown rendering with tables, code blocks, headings
- JSON pretty-print viewer
- CSV to HTML table conversion
- HTML iframe sandbox preview
- PDF viewer via data URL iframe
- Excel viewer powered by Luckysheet
- PPTX viewer powered by pptx-viewer.js
- Static file serving for Luckysheet assets
- PROFILE.md injection for agent capability awareness
- One-click install/uninstall scripts
- Auto config directory detection (.copaw / .qwenpaw)
